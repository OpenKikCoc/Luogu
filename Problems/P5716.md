#  [【深基3.例9】月份天数](https://www.luogu.com.cn/problem/P5716)

## 题意

>   ## 题目描述
>
>   输入年份和月份，输出这一年的这一月有多少天。需要考虑闰年。
>
>   ## 输入格式
>
>   无
>
>   ## 输出格式
>
>   无
>
>   ## 输入输出样例
>
>   **输入 #1**复制
>
>   ```
>   1926 8
>   ```
>
>   **输出 #1**复制
>
>   ```
>   31
>   ```
>
>   **输入 #2**复制
>
>   ```
>   2000 2
>   ```
>
>   **输出 #2**复制
>
>   ```
>   29
>   ```

## 题解



```c++
#include <bits/stdc++.h>
using namespace std;

int y, m;

bool f(int x) {
    return x % 4 == 0 && x % 100 || x % 400 == 0;
}

int month[13] = {0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};

int main() {
    cin >> y >> m;
    
    int t = month[m];
    if (f(y) && m == 2)
        t ++ ;
    cout << t << endl;
    
    return 0;
}
```
