#  [集合求和](https://www.luogu.com.cn/problem/P2415)

## 题意

>   ## 题目描述
>
>   给定一个集合s（集合元素数量<=30），求出此集合所有子集元素之和。
>
>   ## 输入格式
>
>   集合中的元素（元素<=1000）
>
>   ## 输出格式
>
>   和
>
>   ## 输入输出样例
>
>   **输入 #1**复制
>
>   ```
>   2 3
>   ```
>
>   **输出 #1**复制
>
>   ```
>   10
>   ```
>
>   ## 说明/提示
>
>   子集为：
>
>   [] [2] [3] [2 3] 2+3+2+3=10
>
>   保证结果在10^18以内。

## 题解



```c++
#include <bits/stdc++.h>
using namespace std;

using ULL = unsigned long long;

int main() {
    vector<ULL> ve;
    
    int x;
    while (cin >> x)
        ve.push_back(x);
        
    ULL n = ve.size();
    
    ULL res = 0;
    for (auto x : ve)
        res += x * (1 << (n - 1));
    cout << res << endl;
    
    return 0;
}
```
