#  [【深基5.习7】杨辉三角](https://www.luogu.com.cn/problem/P5732)

## 题意

>   ## 题目描述
>
>   给出 n(n≤20)*n*(*n*≤20)，输出杨辉三角的前 n*n* 行。
>
>   如果你不知道什么是杨辉三角，可以观察样例找找规律。
>
>   ## 输入格式
>
>   无
>
>   ## 输出格式
>
>   无
>
>   ## 输入输出样例
>
>   **输入 #1**复制
>
>   ```
>   6
>   ```
>
>   **输出 #1**复制
>
>   ```
>   1
>   1 1
>   1 2 1
>   1 3 3 1
>   1 4 6 4 1
>   1 5 10 10 5 1
>   ```

## 题解



```c++
#include <bits/stdc++.h>
using namespace std;

const int N = 25;

int n;
int g[N][N];

int main() {
    cin >> n;
    
    g[1][1] = 1;
    
    for (int i = 2; i <= n; ++ i )
        for (int j = 1; j <= i; ++ j )
            g[i][j] = g[i - 1][j - 1] + g[i - 1][j];
    
    for (int i = 1; i <= n; ++ i ) {
        for (int j = 1; j <= i; ++ j )
            cout << g[i][j] << ' ';
        cout << endl;
    }
    
    return 0;
}
```
