#  [【深基2.例7】数字反转](https://www.luogu.com.cn/problem/P5705)

## 题意

>   ## 题目描述
>
>   输入一个不小于 100100 且小于 10001000，同时包括小数点后一位的一个浮点数，例如 123.4123.4，要求把这个数字翻转过来，变成 4.3214.321 并输出。
>
>   ## 输入格式
>
>   一行一个浮点数
>
>   ## 输出格式
>
>   一行一个浮点数
>
>   ## 输入输出样例
>
>   **输入 #1**复制
>
>   ```
>   123.4
>   ```
>
>   **输出 #1**复制
>
>   ```
>   4.321
>   ```

## 题解



```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    string s;
    cin >> s;
    reverse(s.begin(), s.end());
    cout << s << endl;
    return 0;
}
```
