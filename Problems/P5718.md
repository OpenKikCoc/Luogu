#  [【深基4.例2】找最小值](https://www.luogu.com.cn/problem/P5718)

## 题意

>   ## 题目描述
>
>   给出 n(n≤100)*n*(*n*≤100) 和 n*n* 个整数 ai(0≤ai≤1000)*a**i*(0≤*a**i*≤1000)，求这 n*n* 个整数中最小值是什么。
>
>   ## 输入格式
>
>   无
>
>   ## 输出格式
>
>   无
>
>   ## 输入输出样例
>
>   **输入 #1**复制
>
>   ```
>   8
>   1 9 2 6 0 8 1 7
>   ```
>
>   **输出 #1**复制
>
>   ```
>   0
>   ```

## 题解



```c++
#include <bits/stdc++.h>
using namespace std;

const int N = 110;

int n;
int a[N];

int main() {
    cin >> n;
    
    int res = 2e9;
    for (int i = 0; i < n; ++ i ) {
        int v;
        cin >> v;
        res = min(res, v);
    }
    cout << res << endl;
    
    return 0;
}
```
