#  [编号](https://www.luogu.com.cn/problem/P1866)

## 题意

>   ## 题目描述
>
>   太郎有N只兔子，现在为了方便识别它们，太郎要给他们编号。兔子们向太郎表达了它们对号码的喜好，每个兔子i想要一个整数，介于1和Maxnumber[i]之间（包括1和Maxnumber[i]）。当然，每个兔子的编号是不同的。现在太郎想知道一共有多少种编号的方法。
>
>   你只用输出答案mod 1000000007即可。如果这是不可能的，就输出0.
>
>   ## 输入格式
>
>   第一行是一个整数N。（1≤N≤50）
>
>   第二行N个整数Maxnumber[i]。（1≤Maxnumber[i]≤1000）
>
>   ## 输出格式
>
>   一个整数
>
>   ## 输入输出样例
>
>   **输入 #1**复制
>
>   ```
>   2
>   5 8
>   ```
>
>   **输出 #1**复制
>
>   ```
>   35
>   ```

## 题解



```c++
#include <bits/stdc++.h>
using namespace std;

using LL = long long;
const int N = 1e4 + 10, MOD = 1e9 + 7;

int n;
int a[N];

int main() {
    cin >> n;
    for (int i = 0; i < n; ++ i )
        cin >> a[i];
    sort(a, a + n);
    
    LL res = 1;
    for (int i = 0; i < n; ++ i ) {
        if (a[i] <= i) {
            // 只有这种情况无解
            cout << 0 << endl;
            return 0;
        }
        res = res * (a[i] - i) % MOD;
    }
    cout << res << endl;
    
    return 0;
}
```
