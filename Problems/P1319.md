#  [压缩技术](https://www.luogu.com.cn/problem/P1319)

## 题意

>   ## 题目描述
>
>   设某汉字由N × N的0和1的点阵图案组成。
>
>   我们依照以下规则生成压缩码。连续一组数值：从汉字点阵图案的第一行第一个符号开始计算，按书写顺序从左到右，由上至下。第一个数表示连续有几个0，第二个数表示接下来连续有几个1，第三个数再接下来连续有几个0，第四个数接着连续几个1，以此类推……
>
>   例如: 以下汉字点阵图案：
>
>   ```
>   0001000
>   0001000
>   0001111
>   0001000
>   0001000
>   0001000
>   1111111
>   ```
>
>   对应的压缩码是： 7 3 1 6 1 6 4 3 1 6 1 6 1 3 7 （第一个数是N ,其余各位表示交替表示0和1 的个数，压缩码保证 N × N=交替的各位数之和）
>
>   ## 输入格式
>
>   一行，压缩码。
>
>   ## 输出格式
>
>   汉字点阵图（点阵符号之间不留空格）。（3<=N<=200）
>
>   ## 输入输出样例
>
>   **输入 #1**复制
>
>   ```
>   7 3 1 6 1 6 4 3 1 6 1 6 1 3 7
>   ```
>
>   **输出 #1**复制
>
>   ```
>   0001000
>   0001000
>   0001111
>   0001000
>   0001000
>   0001000
>   1111111
>   ```

## 题解



```c++
#include<bits/stdc++.h>
using namespace std;

const int N = 210;

int n, c;
int g[N][N];

int main() {
    cin >> n;
    
    bool f0 = true;
    string s;
    while (cin >> c) {
        if (f0)
            s += string(c, '0');
        else
            s += string(c, '1');
        f0 ^= 1;
    }
    
    int count = 0;
    for (auto c : s) {
        cout << c;
        if ( ++ count % n == 0)
            cout << endl;
    }
    return 0;
}
```
