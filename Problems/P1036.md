#  [[NOIP2002 普及组] 选数](https://www.luogu.com.cn/problem/P1036)

## 题意

>   ## 题目描述
>
>   已知 n*n* 个整数 x1,x2,…,xn*x*1,*x*2,…,*x**n*，以及11个整数k*k*(k<n*k*<*n*)。从n*n*个整数中任选k*k*个整数相加，可分别得到一系列的和。例如当n=4,k=3*n*=4,*k*=3,44个整数分别为3,7,12,193,7,12,19时，可得全部的组合与它们的和为：
>
>   3+7+12=223+7+12=22
>
>   3+7+19=293+7+19=29
>
>   7+12+19=387+12+19=38
>
>   3+12+19=343+12+19=34。
>
>   现在，要求你计算出和为素数共有多少种。
>
>   例如上例，只有一种的和为素数：3+7+19=293+7+19=29。
>
>   ## 输入格式
>
>   键盘输入，格式为：
>
>   n,k*n*,*k*(1≤n≤20,k<n1≤*n*≤20,*k*<*n*)
>
>   x1,x2,…,xn(1≤xi≤5000000)*x*1,*x*2,…,*x**n*(1≤*x**i*≤5000000)
>
>   ## 输出格式
>
>   屏幕输出，格式为： 11个整数（满足条件的种数）。
>
>   ## 输入输出样例
>
>   **输入 #1**复制
>
>   ```
>   4 3
>   3 7 12 19
>   ```
>
>   **输出 #1**复制
>
>   ```
>   1
>   ```
>
>   ## 说明/提示
>
>   **【题目来源】**
>
>   NOIP 2002 普及组第二题

## 题解



```c++
#include <bits/stdc++.h>
using namespace std;

const int N = 22;

int n, k;
int a[N];

bool is_prime(int x) {
    if (x < 2)
        return false;
    for (int i = 2; i <= x / i; ++ i )
        if (x % i == 0)
            return false;
    return true;
}

int main() {
    cin >> n >> k;
    for (int i = 0; i < n; ++ i )
        cin >> a[i];
    
    int res = 0;
    for (int i = 0; i < 1 << n; ++ i ) {
        int c = 0, s = 0;
        for (int j = 0; j < n; ++ j )
            if (i >> j & 1)
                c ++ , s += a[j];
        if (c != k)
            continue;
        
        if (is_prime(s))
            res ++ ;
    }
    cout << res << endl;
    
    return 0;
}
```
