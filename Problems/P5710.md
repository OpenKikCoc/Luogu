#  [【深基3.例2】数的性质](https://www.luogu.com.cn/problem/P5710)

## 题意

>   ## 题目描述
>
>   一些数字可能拥有以下的性质：
>
>   -   性质 1：是偶数；
>   -   性质 2：大于 4 且不大于 12。
>
>   小A 喜欢这两个性质同时成立的数字；Uim 喜欢这至少符合其中一种性质的数字；八尾勇喜欢刚好有符合其中一个性质的数字；正妹喜欢不符合这两个性质的数字。
>
>   ## 输入格式
>
>   输入一个数字 x(0≤x≤1000)*x*(0≤*x*≤1000)
>
>   ## 输出格式
>
>   输出这 4 个人是否喜欢这个数字，如果喜欢则输出`1`，否则输出`0`，用空格分隔。
>
>   ## 输入输出样例
>
>   **输入 #1**复制
>
>   ```
>   12
>   ```
>
>   **输出 #1**复制
>
>   ```
>   1 1 0 0
>   ```

## 题解



```c++
#include <bits/stdc++.h>
using namespace std;

bool f1(int x) {
    return x % 2 == 0;
}

bool f2(int x) {
    return x >= 4 && x <= 12;
}

int main() {
    int x;
    cin >> x;
    
    bool a = f1(x), b = f2(x);
    
    cout << (a && b ? 1 : 0) << ' '
         << (a || b ? 1 : 0) << ' '
         << (a ^ b ? 1 : 0) << ' '
         << (!a && !b ? 1 : 0) << endl;
    
    return 0;
}
```
