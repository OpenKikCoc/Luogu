#  [【深基2.例6】字母转换](https://www.luogu.com.cn/problem/P5704)

## 题意

>   ## 题目描述
>
>   输入一个小写字母，输出其对应的大写字母。例如输入 q[回车] 时，会输出 Q。
>
>   ## 输入格式
>
>   无
>
>   ## 输出格式
>
>   无
>
>   ## 输入输出样例
>
>   **输入 #1**复制
>
>   ```
>   q
>   ```
>
>   **输出 #1**复制
>
>   ```
>   Q
>   ```

## 题解



```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    char c;
    cin >> c;
    cout << char(c - 'a' + 'A') << endl;
    
    return 0;
}
```
